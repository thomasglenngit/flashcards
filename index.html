Study and remember these:

JavaScript

JavaScript (or "JS") is a programming language used most often for dynamic client-side scripts on webpages, but it is also often used on the server-side, using packages such as Node.js.

Variables

a symbolic name associated with a value and whose associated value may be changed.

Variable naming conventions

Because JavaScript is case sensitive, letters include the characters "A" through "Z" (uppercase) as well as "a" through "z" (lowercase)

Functions

A function is a code snippet that can be called by other code or by itself, or a variable that refers to the function. When a function is called, arguments are passed to the function as input, and the function can optionally return a value. A function in JavaScript is also an object.

Methods

A method in object-oriented programming (OOP) is a procedure associated with a message and an object. An object consists of data and behavior. The data and behavior comprise an interface, which specifies how the object may be utilized by any of various consumers[1] of the object.

Arguments

An argument is a value (primitive or object) passed as input to a function.

Parameters

A parameter is a named variable passed into a function. Parameter variables are used to import arguments into functions.

Return

The return statement ends function execution and specifies a value to be returned to the function caller.

Chaining methods

You can use optional chaining when attempting to call a method which may not exist. This can be helpful, for example, when using an API in which a method might be unavailable, either due to the age of the implementation or because of a feature which isn't available on the user's device.

Strings

a String has the ability to hold a string or a sequence of characters.

Booleans

a Boolean Data Type can hold only a true or false value at any given time.

NaN

"Not A Number", or "Undefined".

Null

The value null represents the intentional absence of any object value. It is one of JavaScript's primitive values and is treated as falsy for boolean operations.

Alert

The alert role is used to communicate an important and usually time-sensitive message to the user. When this role is added to an element, the browser will send out an accessible alert event to assistive technology products which can then notify the user about it.

Comments

Explanatory text or information embedded in the source code of a computer program.
